Они позволяют передавать данные между горутинами, создавая удобную и эффективную систему для обмена информацией

Канал в Go — это тип данных, предназначенный для передачи значений между горутинами. Он может передавать данные только определённого типа, который называется типом элементов канала.

---

Для создания канала используется встроенная функция **make**:
```go
channel := make(chan int) // Канал типа 'chan int'
```

**Основные операции с каналами:**
- **Отправка (запись)**: передача данных из одной горутины в канал, чтобы другая горутина могла их получить.
- **Получение (чтение)**: извлечение данных из канала в другую горутину.

```go
channel <- num    // отправка данных в канал
num = <- channel   // получение данных из канала
```

---

Чтение данных из канала также может происходить в цикле:

```go
for v := range channel {
    // обработка данных
}
```

В данном случае цикл будет продолжаться до тех пор, пока канал не будет закрыт.

---

Для закрытия канала используется встроенная функция **close**:
```go
close(channel)
```

Закрытие канала сигнализирует о том, что передача данных через этот канал больше не будет происходить. Попытка отправить данные в закрытый канал приведет к панике. Однако чтение из закрытого канала всё ещё возможно, и будут возвращены все оставшиеся значения, которые не были считаны ранее. После того, как все данные будут извлечены, дальнейшее чтение вернёт нулевое значение для типа канала.
