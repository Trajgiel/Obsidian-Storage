Массив - это упорядоченная последовательность элементов **одного типа фиксированной длины**. В Go массив объявляется следующим образом:

```go
var a [3]int
fmt.Println(a) // [0 0 0]
```

Функция `len()` возвращает длину массива.

---

```go
var a [3]int = [3]int{1, 2, 3}
b := [3]int{1, 2, 3}
c := [...]int{1, 2, 3}
d := [3]int{1: 12}

fmt.Println(a) // [1 2 3]
fmt.Println(b) // [1 2 3]
fmt.Println(c) // [1 2 3]
fmt.Println(d) // [0 12 0]
```

Первые два варианта (a и b) интуитивно понятны, но разберем подробнее c и d:

В случае с `c` вместо явного указания длины массива используется многоточие (`...`) в квадратных скобках. Это позволяет компилятору автоматически определить длину массива на основе количества переданных значений. Однако такой подход не считается идиоматичным для Go и не рекомендуется к использованию в [официальной документации](https://golang.org/doc/effective_go.html#arrays).

В примере `d` используется явная инициализация элемента по индексу (`1: 12`). Это позволяет задать значение конкретному элементу, оставляя остальные элементы массива равными нулю (по умолчанию). Индексы могут идти в любом порядке, а некоторые из них можно вообще не указывать.

---

### Сравнение массивов

```go
a := [3]int{1, 2, 3}
b := [3]int{1, 2, 3}
c := [3]int{3, 2, 1}

fmt.Println(a == b) // true
fmt.Println(a == c) // false
```

**Важно помнить**: сравнивать можно только массивы одного типа, то есть одинаковой длины и с одинаковым типом элементов.

---

### Обращение к элементам массива. Индексы

Для работы с элементами массива используются индексы — порядковые номера элементов. Важно помнить, что индексация в Go начинается с нуля: первый элемент массива имеет индекс 0, второй — 1, и так далее. Индекс указывается в квадратных скобках. С его помощью можно как получить значение элемента, так и изменить его:
```go
var numbers [5]int = [5]int{10, 20, 30, 40, 50}    

fmt.Println(numbers[0])  // 10  
fmt.Println(numbers[4])  // 50   

numbers[0] = 99    

fmt.Println(numbers[0])  // 99
```

Поменять местами элементы массива:
```go
array[1], array[2] = array[2], array[1]
```

### range

Go также предоставляет удобный способ перебора элементов массива с помощью ключевого слова `range`. При использовании `range` мы получаем два значения: индекс элемента и его копию.

```go
a := [5]int{5, 10, 15, 20, 25}

for idx := range a {
    fmt.Println(a[idx]) // Вывод значений массива через индекс
}

for _, elem := range a {
    fmt.Println(elem) // Вывод значений напрямую
}
```

Чтобы действительно изменить элементы массива, необходимо обращаться к ним по индексу, а не к их копиям.
