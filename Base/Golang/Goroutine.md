Goroutine - это функция или метод, которая выполняется параллельно с другими горутинами в рамках одной программы на языке Go

Запуск горутины очень прост, достаточно прописать перед функцией ключевое слово "go", например:
```go
func main() {
	go myFunc()
}
```

---

_По умолчанию используется кол-во ядер вашего процессора. Но мы можем изменить количество используемых ядер простой строчкой кода. Приложению будет дана команда перейти на 4 ядра:_

```go
runtime.GOMAXPROCS(4)
```

---

### Блокировка горутин

Горутины могут быть заблокированы в следующих ситуациях:

- **Блокировка при записи:** горутина, отправляющая данные в канал, блокируется, пока другая горутина не получит эти данные.
- **Блокировка при чтении:** горутина, получающая данные из канала, блокируется, пока не появятся новые данные для получения.
