Go - это компилируемый, статически типизированный язык программирования, разработанный компанией Google. Он ориентирован на создание различных приложений, прежде всего веб-сервисов и клиент-серверных решений. Однако Go также предоставляет инструменты для работы с графикой, низкоуровневого программирования и других задач.

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```

`go help doc` - если вы хотите более подробно узнать о работе команды **doc**
`go doc fmt.Println` - если мы хотим узнать подробности о функции **Println()**

```go
go mod init <name> // инициализировать новый модуль в текущем каталоге

go mod download // скачать модули в локальный кеш
go mod edit     // редактировать go.mod из инструментов или скриптов
go mod graph    // напечатать граф требований модуля
go mod tidy     // добавить отсутствующие и удалить неиспользуемые модули
go mod vendor   // делает вендорную копию зависимостей
go mod verify   // проверить зависимости ожидаемого содержания
go mod why      // объяснять, зачем нужны пакеты или модули
```

[Функция](Функции) представляет блок операторов, которые все вместе выполняют какую-то определенную задачу.

[[Строки]] представляют собой неизменяемые последовательности байтов.

[Массив](Массивы) - это упорядоченная последовательность элементов **одного типа фиксированной длины**. 

[Слайс](Слайсы) - это последовательность элементов **одного типа**, но в отличие от массива, его **длина может изменяться**.

[[Maps]] - это структура данных, представляющая собой неупорядоченную коллекцию пар "ключ-значение".

[Структура](Структуры) - это тип данных, который позволяет объединить несколько значений (полей) разных типов в один логический объект.

[Interfaces](Interface) - это типы, которые определяют набор методов. Интерфейсы позволяют создавать абстракции, не зависимые от конкретной реализации.

---

[[Область видимости]] - это участок кода, в пределах которого переменная или другая сущность (функция, тип) доступна и может быть использована.

в Go [ошибки](Обработка%20ошибок) - это значения с типом `error`, возвращаемые функциями, как и любые другие значения.

Функция [[init()]] - это специальная функция, которая используется для **инициализации пакета**.

[[Оператор defer]] позволяет выполнить определенную операцию после выполнения других действий, при этом не важно, где в реальности вызывается эта функция.

[Указатель](Указатели) - значение, которое указывает на адрес другой переменной.

[[Приведение типа]] (или преобразование типов) выполняется явно. Чтобы преобразовать значение из одного типа в другой, нужно явно указать целевой тип в скобках

[Конкурентность и параллелизм](Concurrency,%20Parallelism) — два важных концепта многозадачности, которые часто используются в контексте многопоточного программирования.

[[Goroutine]] - это функция или метод, которая выполняется параллельно с другими горутинами в рамках одной программы на языке Go.

[[Каналы]] позволяют передавать данные между горутинами, создавая удобную и эффективную систему для обмена информацией.
